# NovelReader
A simple novel reader demo using a TTS engine with ggml backend.
Uses work from [CrispASR](https://github.com/CrispStrobe/CrispASR) and [ggml](https://github.com/ggml-org/ggml)

For now it supports [Qwen3-TTS](https://github.com/QwenLM/Qwen3-TTS), [Fish Audio's S2 Pro](https://github.com/fishaudio/fish-speech), and [Higgs TTS](https://github.com/bosonai/Higgs-Audio-v3) (see [LICENSE-HIGGS](LICENSE-HIGGS)).
